# BloxBang
Made by a single guy pulling an all nighter right now writing this readme. This is probably the BEST keyless & free BloxStrike rage script.
The script is up to date as of the 19th of June 2026. This script is only usable on computer platforms, **does not work on mobile!!!**

The script has a nice text based interface. Easy use and no lag!

Features of v1 include:
- Big Heads
- Force 3rd Person
- Auto Jump
- Dark Visuals (in the menu as BloxBang Visuals)
- Player Outlines
- Stealth mode (hides UI)
